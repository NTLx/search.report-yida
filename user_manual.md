## 🎯 系统简介

报告查询系统是一个便捷的在线工具，允许您通过简单的个人信息查询和下载您的报告文件。系统设计简洁易用，无需注册或登录，只需提供您的姓名和手机号码即可查询。

### 主要功能

- **快速查询**: 输入姓名和手机号即可查询相关报告
- **安全下载**: 通过验证后可安全下载您的报告文件
- **多设备支持**: 支持电脑、手机、平板等多种设备访问
- **响应式界面**: 自动适应不同屏幕尺寸，提供最佳浏览体验

### 系统要求

- 现代浏览器（推荐使用Chrome、Firefox、Safari或Edge的最新版本）
- 稳定的网络连接
- 支持JavaScript的浏览器环境

## 🚀 快速入门

### 访问系统

在浏览器中打开系统网址（由系统管理员提供），您将看到简洁的查询界面。

### 基本查询步骤

1. **输入姓名**: 在"姓名"输入框中输入您的完整姓名
2. **输入手机号**: 在"手机号"输入框中输入您的手机号码
3. **点击查询**: 点击"查询报告"按钮
4. **查看结果**: 系统将显示匹配的报告列表
5. **下载报告**: 点击报告名称或下载按钮获取文件

### 示例操作

```
姓名：张三
手机号：13800138000
```

点击"查询报告"按钮后，系统将显示与该信息匹配的所有报告。

## 📖 详细操作指南

### 查询界面说明

系统主界面包含以下元素：

-   **标题区域**: 显示系统名称和简介
-   **查询表单**: 包含姓名和手机号输入框
-   **查询按钮**: 提交查询请求
-   **结果区域**: 显示查询结果和下载链接
-   **主题切换**: 右上角按钮，用于切换深色/浅色模式
-   **GitHub链接**: 左上角按钮，访问项目源代码
-   **页脚信息**: 包含系统版本和联系方式

### 输入信息规范

#### 姓名输入

- 请输入您的完整姓名，确保与系统记录中的姓名完全一致
- 姓名区分大小写，请按照证件上的准确姓名输入
- 支持中文、英文及特殊字符
- 示例：
  - 正确：张三、李四、Wang Xiaoming
  - 错误：张 三、李 四、WANGXIAOMING（大小写不一致）

#### 手机号输入

- 请输入11位手机号码
- 不需要添加国家代码或分隔符
- 示例：
  - 正确：13800138000
  - 错误：+8613800138000、138-0013-8000

### 高级查询选项

系统支持以下高级查询选项（如可用）：

- **日期范围**: 可选择查询特定时间范围内的报告
- **报告类型**: 可筛选特定类型的报告
- **每页显示数量**: 可调整每页显示的报告数量

### 查询结果说明

查询结果页面包含以下信息：

- **报告列表**: 显示匹配的报告文件
- **文件信息**: 包括文件名、创建时间、文件大小等
- **下载链接**: 点击可直接下载报告文件
- **分页控件**: 如结果较多，可翻页查看

### 下载报告

1. 在查询结果中找到您需要的报告
2. 点击报告名称或下载按钮
3. 浏览器将开始下载文件
4. 根据浏览器设置，文件将保存到默认下载位置

#### 下载注意事项

- 报告文件通常为PDF格式，请确保您的设备有相应的阅读软件
- 大文件下载可能需要较长时间，请耐心等待
- 如下载失败，请检查网络连接后重试
- 某些浏览器可能会阻止自动下载，请允许下载操作

## 🎭 常见操作场景

### 场景1：首次查询报告

**目标**: 第一次使用系统查询体检报告

**步骤**:
1. 打开系统网址
2. 在姓名框输入"张明"
3. 在手机号框输入"13912345678"
4. 点击"查询报告"按钮
5. 查看结果列表中的"2023年度体检报告.pdf"
6. 点击报告名称下载文件

### 场景2：查询历史报告

**目标**: 查询过去一年的所有报告

**步骤**:
1. 输入姓名和手机号
2. 在日期范围中选择"过去一年"
3. 点击"查询报告"
4. 浏览所有历史报告
5. 选择需要的报告进行下载

### 场景3：手机端操作

**目标**: 在手机上查询并下载报告

**步骤**:
1. 在手机浏览器中打开系统网址
2. 输入姓名和手机号（可使用数字键盘）
3. 点击"查询报告"按钮
4. 在结果列表中点击所需报告
5. 等待文件下载完成
6. 在手机文件管理器中查找下载的报告

### 场景4：切换深色模式

**目标**: 在夜间使用时切换到深色模式以保护视力

**步骤**:
1. 找到页面右上角的"月亮"图标按钮
2. 点击按钮，界面将瞬间切换为深色背景
3. 如需切回浅色模式，再次点击该按钮（此时图标为"太阳"）

### 场景5：查询失败处理

**目标**: 处理查询无结果的情况

**步骤**:
1. 检查输入的姓名是否正确，注意大小写和空格
2. 确认手机号码是否为11位且无误
3. 尝试使用不同的姓名格式（如全名vs简称）
4. 联系系统管理员确认信息是否已录入系统
5. 如确认信息无误但仍无法查询，请联系技术支持

## 🔧 问题排查

### 常见问题及解决方案

#### 问题1：查询无结果

**可能原因**:
- 输入的姓名与系统记录不一致
- 手机号码输入错误
- 系统中尚未录入您的报告信息

**解决方案**:
1. 仔细检查姓名拼写，注意大小写和空格
2. 确认手机号码是否为11位且无误
3. 尝试使用不同的姓名格式
4. 联系报告发放机构确认信息是否已录入系统

#### 问题2：下载失败

**可能原因**:
- 网络连接不稳定
- 浏览器阻止了下载
- 文件过大导致超时

**解决方案**:
1. 检查网络连接，尝试刷新页面后重新下载
2. 检查浏览器下载设置，允许网站下载文件
3. 尝试使用其他浏览器下载
4. 如文件过大，尝试在网络状况良好时下载

#### 问题3：页面显示异常

**可能原因**:
- 浏览器版本过低
- JavaScript被禁用
- 缓存问题

**解决方案**:
1. 更新浏览器至最新版本
2. 检查浏览器设置，确保JavaScript已启用
3. 清除浏览器缓存后重新访问
4. 尝试使用无痕/隐私模式访问

#### 问题4：手机端操作不便

**可能原因**:
- 页面未完全加载
- 触摸操作不灵敏
- 输入法问题

**解决方案**:
1. 等待页面完全加载后再进行操作
2. 尝试刷新页面或重新打开
3. 使用系统输入法而非第三方输入法
4. 如问题持续，尝试使用电脑访问

### 错误信息解读

| 错误信息 | 含义 | 解决方案 |
|---------|------|----------|
| "未找到匹配的报告" | 系统中无与输入信息匹配的报告 | 检查输入信息或联系管理员 |
| "网络连接失败" | 无法连接到服务器 | 检查网络连接后重试 |
| "请求超时" | 服务器响应时间过长 | 稍后重试或联系技术支持 |
| "文件下载失败" | 无法下载报告文件 | 检查网络和浏览器设置 |

### 性能优化建议

1. **使用最新浏览器**: 确保使用浏览器的最新版本以获得最佳性能
2. **清理缓存**: 定期清理浏览器缓存以提高加载速度
3. **关闭不必要的标签页**: 减少浏览器资源占用
4. **使用稳定网络**: 在网络状况良好时使用系统
